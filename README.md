
# 牛牛游戏项目

## 启动方式
1. 本地运行：
```bash
npm install
npm start
```

2. 云端部署（Render）：
- 上传代码到GitHub
- Render新建 Web Service，连接仓库自动部署
- 部署成功后访问：`https://xxx.onrender.com`

管理员后台访问路径：`https://xxx.onrender.com/admin.html`
